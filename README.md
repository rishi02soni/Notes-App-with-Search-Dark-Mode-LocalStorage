# 📝 Notes App (Vanilla JavaScript)

A modern, fast, and lightweight **Notes Management Web Application** built using **pure JavaScript (no frameworks)**.
This app allows users to create, search, and manage notes with a clean UI and persistent storage.

---

## 🚀 Demo Preview

![App Demo](https://media.giphy.com/media/l0MYt5jPR6QX5pnqM/giphy.gif)

---

## 📌 Features

* 📝 Add and delete notes
* 🔍 Real-time search functionality
* 🌙 Dark / Light mode toggle
* 💾 LocalStorage persistence
* ⚡ Instant UI updates
* 🎯 Responsive and minimal design

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript (ES6)
* **Storage:** Browser LocalStorage
* **Architecture:** Event-driven DOM manipulation

---

## 📂 Project Structure

```bash
notes-app/
│── index.html
│── style.css
│── script.js
```

---

## ⚙️ How It Works

1. User writes a note in the textarea
2. Note is stored in an array and saved to LocalStorage
3. Notes are dynamically rendered on screen
4. Search filters notes in real-time
5. Theme toggle switches UI mode instantly
6. Data persists even after page reload
